# CocoaCraft Chocolates – Shipment Performance Analytics Dashboard

## Project Overview

This project showcases an interactive Power BI dashboard developed to analyze shipment performance for CocoaCraft Chocolates over a 13-month period from **01-Feb-2023 to 28-Feb-2024**.

The dashboard provides business stakeholders with a centralized view of shipment activity, sales performance, profitability, product performance, and salesperson contribution across multiple geographies.

## Business Objectives

* Monitor shipment performance over time
* Track sales, cost, profit, and profitability trends
* Analyze salesperson performance
* Compare performance across geographies
* Evaluate product category contribution
* Support data-driven decision making through interactive reporting

---

## Key Features

* KPI cards for Sales, Boxes, Shipments, Cost, Profit, and Profit %
* Month-over-Month performance analysis
* Interactive geography filtering
* Salesperson performance ranking
* Shipment trend analysis
* Dynamic measure switching using Field Parameters
* Bookmark-based navigation for optimized dashboard space
* Profitability and operational performance monitoring

---

## Data Model

![Data Model](screenshots/datamodel.png)

The solution follows a dimensional modeling approach consisting of:

* Fact Table: Shipments
* Dimension Tables:

  * Products
  * People
  * Calendar
  * Locations
* Dedicated Measures Table
* Field Parameter Table for dynamic metric selection
